# 📱 漢字テストジェネレーター - PWAアプリのご案内

**プログラミングのKOBEYA**  
**小学生向け漢字学習アプリ**

---

## 🎯 このアプリでできること

### 📝 プリント生成
- 学年別の漢字テスト作成（小1〜小6）
- 10問/20問から選択可能
- 読みテスト・書きテスト対応
- 印刷機能付き

### ✏️ 手書き練習
- タッチデバイスで漢字の練習
- 漢字をなぞって書き順を学習
- 消しゴム・ペン太さ調整機能
- 練習モード・テストモード切替

### 📖 書き順確認
- 練習中に書き順をすぐに確認
- ワンタップでWeb検索

## 📱 アプリのインストール方法

### iPhone/iPad の場合

1. **Safari で以下のURLを開く**
   ```
   https://kanji-test-generator.pages.dev/
   ```

2. **画面下部の共有ボタン（↑）をタップ**

3. **「ホーム画面に追加」を選択**

4. **「追加」をタップ**

5. **ホーム画面にアイコンが追加されます**

6. **アイコンをタップしてアプリを起動！**

---

### Android の場合

1. **Chrome で以下のURLを開く**
   ```
   https://kanji-test-generator.pages.dev/
   ```

2. **画面上部に「アプリをインストール」バナーが表示されます**

3. **「インストール」をタップ**

または：

- メニュー（⋮）→「アプリをインストール」→「インストール」

4. **ホーム画面にアイコンが追加されます**

5. **アイコンをタップしてアプリを起動！**

---

### PC (Chrome/Edge) の場合

1. **ブラウザで以下のURLを開く**
   ```
   https://kanji-test-generator.pages.dev/
   ```

2. **アドレスバー右側の「インストール」アイコン（⊕）をクリック**

3. **「インストール」をクリック**

4. **デスクトップにショートカットが作成されます**

5. **ショートカットをクリックしてアプリを起動！**

## 🎨 使い方

### 1️⃣ プリント生成
1. ホーム画面から「漢字テストジェネレーター」アイコンをタップ
2. 学年と問題数を選択
3. 「プリント生成」をタップ
4. 印刷またはスクリーンショット

### 2️⃣ 手書き練習
1. ホーム画面から「漢字テストジェネレーター」アイコンをタップ
2. 学年と問題数を選択
3. 「手書き練習へ」をタップ
4. 漢字をなぞって練習

### 3️⃣ 書き順確認
- 練習中に「📖 書き順」ボタンをタップ
- 書き順を確認できます
